개인정보처리방침

SMS 전달 (SMS Forwarder)  |  최종 수정일: 2025년 1월 1일

요약 안내 본 앱은 사용자가 직접 설정한 규칙에 따라 수신 SMS·알림을 지정 번호로 전달하는 자동화 도구입니다. 앱이 수집하는 모든 정보는 기기 내에서만 처리되며, 개발자 서버로 전송하거나 광고·분석 목적으로 사용하지 않습니다.
1

수집하는 정보 및 목적

앱은 핵심 기능 제공을 위해 아래 정보를 처리합니다. 모든 처리는 사용자 기기 내에서만 이루어지며, 외부 개발자 서버로 전송하지 않습니다.

정보 유형 수집 내용 처리 목적
SMS 내용 수신 SMS의 발신번호·본문 사용자 설정 규칙과 비교 후, 조건에 맞으면 지정 번호로 전달
앱 알림 앱 이름, 알림 제목·본문 사용자 설정 규칙과 비교 후, 조건에 맞으면 지정 번호로 전달
연락처 이름 발신번호에 대응하는 저장된 이름 "[송신 번호 : xxx (이름)]" 형식으로 전달 문자 앞에 추가 (해당 옵션 선택 시에만)
부재중 전화 발신번호, 통화 시각 부재중 전화 규칙에 따라 지정 번호로 알림 전달
설치된 앱 목록 앱 이름·패키지명 알림 전달 규칙 생성 시 앱 선택 목록 표시 (기기 내 처리, 서버 미전송)
네트워크 정보 현재 연결된 Wi-Fi SSID 지정 Wi-Fi 연결 시 PC 연결 서비스 자동 시작 (옵션 설정 시에만)
⚠️ 전달 대상이 텔레그램인 경우: 메시지 내용이 사용자가 직접 설정한 텔레그램 봇 API를 통해 Telegram 서버(api.telegram.org)로 전송됩니다. 이 외에 개발자가 운영하는 별도 서버로는 어떠한 데이터도 전송되지 않습니다.
2

요청하는 권한 및 사용 목적

SMS 관련 권한

RECEIVE_SMS READ_SMS SEND_SMS

수신 SMS를 감지하고 규칙에 맞는 경우 지정 번호로 전달하기 위해 사용합니다. SMS 내용은 기기 내 규칙 매칭 후 즉시 파기되며, 개발자 서버에 저장되지 않습니다.

알림 접근 권한

BIND_NOTIFICATION_LISTENER_SERVICE

설정 메뉴 → 알림 접근에서 사용자가 직접 허용해야 하며, 사용자가 설정한 앱의 알림만 처리합니다. 앱 자체 알림 및 순수 시스템 앱 알림은 제외됩니다.

통화 기록 권한

READ_CALL_LOG READ_PHONE_STATE

부재중 전화 규칙 기능에서 발신번호 감지 및 확인을 위해 사용합니다. 통화 기록은 기기 내에서만 처리되며 외부로 전송하지 않습니다.

연락처 권한

READ_CONTACTS

"[송신 번호 : xxx (이름)]" 옵션을 켰을 때, 발신번호에 저장된 이름을 조회합니다. 전달 메시지에 이름을 포함할 목적으로만 사용되며 연락처 목록 전체를 수집하지 않습니다.

네트워크 / 인터넷 권한

INTERNET ACCESS_WIFI_STATE ACCESS_NETWORK_STATE

텔레그램 메시지 전송(사용자 설정 시), PC 연결 웹 서버 운영, Wi-Fi 자동 연결 감지에 사용합니다.

앱 목록 조회 권한

QUERY_ALL_PACKAGES

알림 규칙 생성 화면에서 설치된 앱 목록을 표시하기 위해 사용합니다. 조회된 앱 목록은 기기 밖으로 전송되지 않습니다.

기타 권한

RECEIVE_BOOT_COMPLETED FOREGROUND_SERVICE POST_NOTIFICATIONS

기기 재시작 후 자동 서비스 복구, PC 연결 서비스의 포그라운드 실행, 서비스 상태 알림 표시에 사용합니다.

데이터 보관 및 삭제

데이터 종류 보관 위치 보관 기간 / 삭제 방법
수신된 알림 내역 기기 내부 데이터베이스 최근 100건만 유지 (자동 삭제) · 알림 탭에서 개별/전체 삭제 가능
전달 규칙 설정 기기 내부 데이터베이스 사용자가 직접 삭제하거나, 앱 삭제 시 함께 삭제
웹 서비스 설정
(비밀번호, 포트, 봇 토큰)		 기기 내부 SharedPreferences 앱 설정 초기화 또는 앱 삭제 시 삭제
SMS 내용 · 알림 본문 규칙 매칭 후 메모리에서 처리 전달 완료 후 즉시 파기 (별도 저장 없음)

앱 데이터 전체 삭제 방법

  • Android 설정 → 앱 → SMS 전달 → 저장 공간 → 데이터 지우기
  • 앱 삭제 시 모든 로컬 데이터가 함께 삭제됩니다
4

제3자 제공 및 외부 서비스

앱은 사용자가 직접 설정하지 않는 한 어떠한 외부 서버에도 데이터를 전송하지 않습니다. 사용자 설정에 따라 아래 외부 서비스와 연동될 수 있습니다.

외부 서비스 전송 조건 전송 내용 해당 정책
Telegram Bot API
(api.telegram.org)		 규칙에 텔레그램 채팅 ID와 봇 토큰을 설정한 경우에만 전달 규칙에 매칭된 메시지 내용 Telegram 개인정보처리방침
이동통신망 (SMS) SMS 전달 규칙이 발동된 경우 처리된 메시지 본문 사용자의 통신사 약관 적용

개발자 운영 서버, 광고 네트워크, 분석 서비스로의 데이터 전송은 없습니다.

5

PC 연결 웹 서버 기능

PC 연결 기능은 앱 내에서 경량 HTTP 서버를 실행하여 같은 네트워크(Wi-Fi)에 연결된 PC 브라우저로 앱을 관리할 수 있게 합니다.

  • 서버는 기기 내부에서만 실행되며 인터넷을 통해 외부에서 접근할 수 없습니다
  • 사용자가 설정한 비밀번호로 로그인 후 접근 가능합니다
  • 비밀번호는 기기 내 암호화된 저장소에 보관됩니다
  • 서버를 통해 조회 가능한 데이터(규칙, 알림 내역)는 기기 내 앱 데이터와 동일합니다
  • 서비스 시작/중지는 사용자가 직접 제어합니다
6

아동 및 미성년자 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 아동이 앱을 사용하고 있다고 판단되는 경우 아래 연락처로 알려주시면 즉시 조치하겠습니다.

7

사용자의 권리

사용자는 아래 권리를 행사할 수 있습니다.

  • 접근권: 앱이 처리 중인 데이터를 앱 내 화면(규칙 목록, 알림 탭)에서 직접 확인
  • 삭제권: 앱 내 데이터 삭제 기능 또는 Android 설정의 데이터 지우기를 통해 삭제
  • 처리 거부권: 앱 권한 설정에서 개별 권한을 취소할 수 있으며, 일부 기능이 제한될 수 있습니다
  • 알림 수집 중단: 앱 내 알림 탭의 수집 토글로 즉시 중단 가능
8

보안 조치

  • 앱 내 모든 데이터는 Android 앱 샌드박스 내에 저장되어 다른 앱이 접근할 수 없습니다
  • 텔레그램 전송은 HTTPS(TLS)로 암호화됩니다
  • PC 연결 서비스는 동일 로컬 네트워크에서만 접근 가능하며 비밀번호 보호를 제공합니다
  • 앱 내에 저장되는 봇 토큰, 비밀번호 등 민감한 설정값은 Android SharedPreferences에 보관됩니다
9

개인정보처리방침 변경

본 방침이 변경되는 경우 앱 업데이트 또는 이 페이지를 통해 공지합니다. 중요한 변경사항이 있을 경우 앱 내 알림을 통해 별도로 안내합니다. 변경된 방침은 공지일로부터 7일 후 효력이 발생합니다.
